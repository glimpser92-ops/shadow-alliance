# Shadow Alliance Classroom

교사용 화면 하나로 진행하고, 학생은 QR 코드로 접속해 비밀 팀과 랜덤 닉네임을 받은 뒤 숫자를 제출하는 교실용 협상 게임입니다.

## 실행

```powershell
npm install
npm start
```

서버가 시작되면 교사용 주소와 같은 와이파이 접속 주소가 터미널에 표시됩니다. 교사는 표시된 주소로 접속해 `방 열기`를 누르고, 학생은 QR 코드로 입장합니다.

## 와이파이가 달라도 쓰기

방법은 두 가지입니다.

1. 클라우드 서버에 배포합니다.

   Node.js 웹 서버를 지원하는 곳에 이 폴더를 올리고 아래 명령으로 실행하면 됩니다.

   ```powershell
   npm install
   npm start
   ```

   교사가 배포된 공개 주소로 접속해 방을 열면 QR 코드도 공개 주소로 만들어져 학생들이 어디서든 접속할 수 있습니다.

2. 선생님 PC를 그대로 쓰고 터널 주소를 연결합니다.

   Cloudflare Tunnel, ngrok 같은 터널 도구로 `3000`번 포트를 공개 주소에 연결한 뒤, 아래처럼 공개 주소를 지정하고 서버를 시작합니다.

   ```powershell
   $env:PUBLIC_URL="https://your-public-url.example"
   npm start
   ```

   `PUBLIC_URL`이 있으면 교사용 화면을 `localhost`로 열어도 학생 QR은 공개 주소 기준으로 생성됩니다.

## 구성

- 교사용 방 개설, QR 코드, 입장 현황
- 학생 랜덤 닉네임 및 블랙/화이트 자동 배정
- 5라운드 기본 진행, 라운드마다 설정 범위 안에서 지령 자동 생성
- 5분 협상 타이머, 일시정지, 30초 증감, 조기 결과 공개
- 1~100 숫자 제출 및 라운드 중 재제출
- 승리 팀 숫자만 익명 공개, 패배 팀 숫자 비공개
- 승리 팀 높은 숫자 상위 n명 랜덤 컷 페널티, 나머지 10,000 세력 비율 배분
- 라운드 종료 후 개인 획득 세력 표시
- 학생 화면 비밀 정보는 누르고 있는 동안만 공개
- 최종 시상대, 전체 순위, 진영 명단 공개

## 배포

일반 Node.js 서버로 동작합니다. Render, Railway, Fly.io, 학교 내부 PC, NAS 등에서 `npm install` 후 `npm start`로 실행할 수 있습니다. 외부 배포에서는 환경 변수 `PORT`를 플랫폼이 제공하는 값으로 자동 사용합니다. 프록시나 터널 뒤에서 실행할 때는 `PUBLIC_URL`을 공개 주소로 지정하면 QR 코드가 그 주소로 생성됩니다.
